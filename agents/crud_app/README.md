# CRUD App

This is a basic CRUD (Create, Read, Update, Delete) application built with Flask, SQLite, and HTML/CSS.

## Features

*   Create new messages
*   Read all messages
*   Update existing messages
*   Delete messages

## Usage

1.  Run the application using `python app.py`
2.  Open a web browser and navigate to `http://localhost:5000`
3.  Create, read, update, and delete messages as needed
