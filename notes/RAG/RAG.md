# Basic RAG:
- ***Indexing*** -> PDF File/Data source -> Chunking -> Embeddings -> Store in vector db
- ***Retrieval*** -> User query -> Embeddings -> Similarity Search -> Retrieve relevant chunks
- ***Generation*** -> Generate the answer from the query and the retrieved relevant chunks
